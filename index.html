What is Prosmosou?

Prosmosou provides easy-to-use installer scripts for setting up your environment on any operating system. Whether you're on Linux, Windows, Mac, or our new "No OS" system, you can get started quickly and safely.

Features

  • Supports all major operating systems
  • Easy customization and linking of installers
  • "No OS" system: run your base OS from a live USB, keep your data on a separate drive
  • One setup script for a clean, ready-to-go system every time you boot
  • Safer and more reliable â€” prevents clutter and keeps your system fresh

Quick Start Example

To install a cursor theme on Linux, just run:

./install-cursor.sh

This will fetch the latest version, install it, and set up everything you need automatically.

How the "No OS" System Works

  1. Boot from a live install USB drive
  2. Access your data USB drive
  3. Customize and link your installers into one setup script
  4. Run the setup script on first boot â€” your system is ready to go!
  5. When done, power off. Next boot starts fresh, keeping your system clean and secure.
